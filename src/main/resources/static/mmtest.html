مبانی HTML

HTML زبان استاندارد برای ایجاد صفحات وب است. HTML مخفف Hyper Text Markup Language می‌باشد.

ساختار اصلی یک سند HTML

<!DOCTYPE html> <html> <head> <title>عنوان صفحه</title> </head> <body> <h1>این یک عنوان است</h1> <p>این یک پاراگراف است.</p> </body> </html>

ویژگی‌های مهم

  • id: شناسه یکتا برای المان
  • class: کلاس برای استایل‌دهی
  • style: استایل inline
  • src: آدرس منبع (برای تصاویر)
  • href: آدرس لینک
